# Responsive Python GUI Calculator

A sleek, desktop calculator built with Python and Tkinter. This project focuses on clean code practices, featuring a fully responsive user interface that automatically scales with window resizing and robust error handling to prevent application crashes.

## 🚀 Features

- **Responsive Layout:** The display and buttons dynamically auto-scale and adjust when you resize or maximize the application window.
- **DRY Architecture:** Built using a single optimized loop to render all buttons, keeping the codebase compact and maintainable.
- **Algorithmic Grid Mapping:** Translates a flat list of button labels into a perfect 4x4 layout using mathematical floor division (`//`) and modulo (`%`) operators.
- **Crash-Proof Logic:** Implements safe arithmetic expression evaluation via `try/except` blocks to handle exceptions like division by zero and bad syntax gracefully.

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **GUI Library:** Tkinter (Python's standard GUI package)
